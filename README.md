# text.frittenburger.de
WebService for analysing text

![text.frittenburger.de](webservice.png)

# Quick Start
```
mvn package
java -jar target/NLP-Test-0.0.1-SNAPSHOT-jar-with-dependencies.jar 
```



## Or use docker
```
docker run -it --rm -p 80:4567 frittenburger/text
```
